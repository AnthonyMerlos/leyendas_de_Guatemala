25 Etiquetas de HTML

1. Etiqueta p: Define un parrafo de texto.


2. Etiqueta tittle: Se usa para definir el titulo de una pagina.


3. Etiqueta body: Representa el contenido visible de la pagina web.


4. h1-h6: Etiquetas de encabezado que indican la jerarquia de titulo.


5. a: Crea un enlace de pagina.


6. img: Inserta una imagen en la pagina.


7. ul: Define una lista desordenada.


8. ol: Define una lista ordenada.


9. li: Define un elemento de una lista (•).


10. table: Crea un tabla en la pagina.


11. tr: reprenseta una fila dentro de la tabla.


12. td: define una celda dentro de una fila de la tabla.


13. th: Define una celda de encabezado en una tabla.


14. div: Crea un contenedor generico para estructurar el contenido.


15. span: Permite crear stilo a una parte del texto.


16. br: inseta un salto de linia en el texto.


17. hr: Crea una linia horizontal para separa contenido


18. form: Define un formulario para la recoleccion de datos del usuario.


19. menu: Esta etiqueta se utiliza para definir un menu de comandos.


20. dialog: Introducida en html:5, Permite crear dialogos modales.


21. cite: Sirve para citar el titulo de una obra.


22. figure: Se utiliza para encapsulizar contenido independiente.


23. figcaption: Esta se utiliza dentro de figure para añadir una leyenda.


24. mark: Se utiliza para resaltar texto relevante.


25. script: Permite incorporar codigo Javascript en los documentos de html.
